Для корректной работы библиотек требуется Python 3.11.

1. В корне проекта запустите:
.\installation\run_all_install.ps1

2. Запустите проверку функций на верных или неверных данных:
python tools/runners_incorrect.py
python tools/runners_normal.py

3. Запустите анализ полученных данных:
python experiments/analysis.py
